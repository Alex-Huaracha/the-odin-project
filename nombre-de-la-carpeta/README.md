# JavaScript Linked List

Simple singly-linked list implementation in ES6 modules.

## Features

- Node with `value` and `nextNode`
- LinkedList methods: `append`, `prepend`, `size`, `head`, `tail`, `at`, `pop`, `contains`, `find`, `toString`
- Extra: `insertAt`, `removeAt`

## Requirements

- Node.js v22+ (ES modules support)

## Usage

1. Install Node v22 LTS or newer.
2. Run the example:

```bash
node main.js
```

## Files

- Node.js — Node class (value, nextNode)
- LinkedList.js — LinkedList implementation
- main.js — usage examples
