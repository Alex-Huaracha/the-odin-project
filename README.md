# The Odin Project

## Tic Tac Toe

This is a simple Tic Tac Toe game.

## Features

- Play Tic Tac Toe in your browser
- Enter custom player names
- Interactive game board with turn display
- Detects wins and ties
- Restart button to play again

## Technologies

- HTML
- CSS
- JavaScript (modular, factory functions, module pattern)
