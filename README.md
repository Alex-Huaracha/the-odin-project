# Recursion JS

This repository contains simple JavaScript implementations of classic recursive and iterative algorithms:

## Files

- **fibonacci.js**
  Includes two functions:

  - `fibs(n)`: Returns an array of the first `n` Fibonacci numbers using iteration.
  - `fibsRec(n)`: Returns an array of the first `n` Fibonacci numbers using recursion.

- **mergeSort.js**
  Implements the recursive merge sort algorithm:
  - `mergeSort(arr)`: Returns a sorted array using the merge sort methodology.

## Usage

Run any file with Node.js:

```bash
node fibonacci.js
node mergeSort.js
```

Each file includes example outputs for testing.
