# Personal Homepage

This is a responsive personal homepage built with HTML, CSS, and JavaScript, using Webpack for asset management and bundling.

## Features

- Mobile-first, fully responsive design
- Dynamic project section (projects managed via JavaScript)
- SVG icons for navigation and social links
- Dark theme support
- Clean, accessible layout

## Technologies

- HTML5
- CSS3 (with custom reset)
- JavaScript (ES Modules)
- Webpack

## About

This project is part of the curriculum for [The Odin Project](https://www.theodinproject.com/), designed to practice modern web development skills and portfolio building.

## Usage

1. Clone the repository
2. Run `npm install`
3. Start the development server with `npm run dev`
4. Build for production with `npm run build`

Feel free to customize and use this template for your own portfolio!
