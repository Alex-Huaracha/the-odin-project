A simple web calculator built with HTML, CSS, and JavaScript.

### Keyboard shortcuts:

- Numbers: `0-9`
- Operators: `+`, `-`, `*`, `/`
- Calculate: `Enter` or `=`
- Clear: `Escape` or `C`
- Delete: `Backspace`
- Decimal: `.`
