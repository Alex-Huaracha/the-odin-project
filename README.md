# Battleship Game

A classic Battleship game built with vanilla JavaScript as part of The Odin Project curriculum.

## Features

- **10x10 grid** with traditional ship placement
- **Smart AI opponent** with hunt/target/destroy strategy
- **Manual or random** ship placement
- **Visual feedback** for hits, misses, and sunk ships
- **Responsive design** for desktop and mobile

## Ship Fleet

- 1x Carrier (5 cells)
- 1x Battleship (4 cells)
- 1x Cruiser (3 cells)
- 1x Submarine (3 cells)
- 1x Destroyer (2 cells)

## How to Play

1. **Place your ships** - Click on your board or use "Random Placement"
2. **Attack enemy waters** - Click on the enemy board to attack
3. **Sink all ships** to win

## Tech Stack

- Vanilla JavaScript (ES6 modules)
- CSS3 with modern features
- Webpack for bundling
- Jest for testing
