# Mini Message Board

A simple message board web application built with Node.js and Express. This project is part of [The Odin Project](https://www.theodinproject.com) curriculum.

## Features

- View all messages on the homepage
- Create new messages with a form
- View individual message details
- Input validation with Zod
- Server-side rendering with EJS

## Installation

```bash
npm install
```

## Usage

Start the server:

```bash
npm run dev
```

Visit `https://suitable-kay-alexs-projects-480c4f3c.koyeb.app/` in your browser.

## Technologies

- Node.js
- Express
- EJS
- Zod
